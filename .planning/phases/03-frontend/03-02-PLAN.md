Create `las_frontend/src/views/CogTestSessionView.vue` as a single-file component with `<template>`, `<script setup lang="ts">`, and `<style scoped>`.

**Script setup:**
- Import `useCogTestStore` from `@/stores/cogTest`
- Import `useI18n` from `vue-i18n`, `useRouter` from `vue-router`, `ref`, `nextTick`, `watch`, `onMounted` from `vue`
- Get store instance: `const store = useCogTestStore()`
- Create `userInput = ref('')` for the text input
- Create `messagesContainer = ref<HTMLElement | null>(null)` for auto-scroll
- On mount: if `store.status === 'idle'` and no sessionId, redirect to `/cog-test` (user navigated directly without starting a session)
- If store already has an active session (status is streaming/waiting), just render — do NOT call startSession again

**Auto-scroll:** Watch `store.messages` deeply — on change, `nextTick(() => messagesContainer.value?.scrollTo({ top: messagesContainer.value.scrollHeight, behavior: 'smooth' }))`

**Submit handler:** `submitTurn()` — guard on `!userInput.value.trim()`, call `store.submitUserTurn(userInput.value.trim())`, clear userInput

**Template structure:**
```
div.cog-test-session
  div.session-header
    button.btn.btn-secondary @click="$router.push('/cog-test')" → t('common.back')
    h1 → store.concept
    span.round-badge → t('cogTest.round') + " " + store.currentRound + "/" + store.maxRounds

  div.status-bar (v-if connecting/error/completed)
    div.connecting (v-if status==='connecting') → t('cogTest.connecting')
    div.error (v-if store.error) → store.error
    div.completed (v-if status==='completed') → t('cogTest.sessionComplete')

  div.messages-container (ref="messagesContainer")
    div.message (v-for="(msg, i) in store.messages" :key="i" :class="[msg.role, { streaming: msg.streaming }]")
      span.role-label → roleLabel(msg.role)  // helper: guide→t('cogTest.guide'), challenger→t('cogTest.challenger'), user→t('cogTest.you')
      div.message-content → msg.content
        span.cursor (v-if="msg.streaming") → "|"

  div.input-area
    button.btn.btn-stop (v-if="store.status==='streaming' || store.status==='waiting'") @click="store.stopSession()" → t('cogTest.stopAndDiagnose')
    form.input-form (@submit.prevent="submitTurn" v-if="store.status !== 'completed' && store.status !== 'stopped'")
      input (v-model="userInput" :disabled="store.status !== 'waiting'" :placeholder="t('cogTest.inputPlaceholder')")
      button.btn.btn-primary (type="submit" :disabled="store.status !== 'waiting' || !userInput.trim()") → t('cogTest.send')
```
