Implementation structure:

**`async def run(self, db: AsyncSession) -> AsyncGenerator`**
```
yield session_start event
while not scheduler.is_session_complete and not _stop_event.is_set():
    role = scheduler.current_agent()
    yield turn_start event

    # Run agent turn with retry for Socratic contract
    parsed, tokens_yielded = await _run_agent_turn_with_retry(db, role)

    # yield tokens (already yielded inside _run_agent_turn via queue/buffer)
    # yield turn_complete
    scheduler.advance()

    if scheduler just completed a round (agent flipped back to guide):
        yield round_complete
        await _save_snapshot(db, round_number, round_scores)

yield session_complete
await _save_snapshot(db, round_number=None, ...)  # final aggregated snapshot
unregister_engine(self.session_id)
```

**`async def _run_agent_turn(self, db, role) -> tuple[AgentOutput, list[str]]`**
- Select system_prompt and temperature based on role
- Build messages list: self.history (full, no sliding window)
- Add warning log if len(self.history) > 20
- Accumulate tokens in buffer list
- Yield token events as they arrive
- In `finally` block: call parse_agent_output(full_response), persist turn
- Return (parsed, buffer)

**Socratic contract retry logic (wraps _run_agent_turn):**
```python
for attempt in range(3):  # 0, 1, 2
    parsed, tokens = await _run_agent_turn(db, role)
    if not _violates_socratic_contract(parsed.dialogue_text):
        break
    if attempt < 2:
        wait = (2 ** attempt) + random.uniform(0, 0.5)
        await asyncio.sleep(wait)
    else:
        # Log violation, proceed anyway — don't break session
        logger.warning(f"Socratic contract violated after 2 retries, session={self.session_id}")
```

**`async def _persist_turn(self, db, role, round_number, parsed: AgentOutput, raw_text: str)`**
- Create CogTestTurn row (turn_index=self._turn_index, role, dialogue_text=parsed.dialogue_text, analysis_json=raw_text, understanding_level=parsed.understanding_level, round_number)
- For each blind_spot in parsed.blind_spots: create CogTestBlindSpot row
- Append agent message to self.history: `{"role": "assistant", "content": parsed.dialogue_text}`
- Increment self._turn_index
- db.add() all objects, await db.commit()

**`async def _save_snapshot(self, db, round_number, round_scores)`**
- Calculate score: if round_number is not None, use calculate_round_score for that round; if None (final), use aggregate_session_score(all_round_scores)
- Count blind spots from self.history context or query DB
- Create CogTestSnapshot(session_id, round_number, understanding_score=str(score), blind_spot_count, snapshot_json=json.dumps summary)
- db.add(), await db.commit()

**Stop handling:**
- Wrap the entire run() body in try/except asyncio.CancelledError + finally
- In finally: if session not already complete, yield session_complete with status="stopped", save final snapshot

**User turn handling:**
- After each agent turn_complete, the engine WAITS for user input via `submit_user_turn(text)` before proceeding
- Add `_user_input_queue: asyncio.Queue = asyncio.Queue()` to __init__
- `async def submit_user_turn(self, text: str)`: puts text on queue, appends to history as `{"role": "user", "content": text}`

from fastapi import Query

async def get_current_user_from_query(
    token: str = Query(..., description="JWT access token (for SSE — EventSource cannot send headers)"),
    db: AsyncSession = Depends(get_db),
) -> User:
    """
    JWT auth via query param. Used by SSE endpoints because
    EventSource API cannot send custom Authorization headers.
    """
    # Reuse the same token validation logic as get_current_user
    # but read token from ?token= query param instead of Authorization header
    from app.core.auth import decode_access_token  # adjust import path as needed
    payload = decode_access_token(token)
    if payload is None:
        raise HTTPException(status_code=401, detail="Invalid or expired token")
    user_id = payload.get("sub")
    if not user_id:
        raise HTTPException(status_code=401, detail="Token missing subject")
    user = await db.get(User, user_id)
    if user is None:
        raise HTTPException(status_code=401, detail="User not found")
    return user
