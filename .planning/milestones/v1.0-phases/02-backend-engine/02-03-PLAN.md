**POST /cog-test/sessions** — Create session
- Auth: `Depends(get_current_user)` (standard header auth)
- Body: `{"concept": str, "max_rounds": int = 3}`
- Logic:
  1. Query DB for any `active` CogTestSession for this user
  2. If found: get_engine(existing.id) and call await engine.stop() if engine exists; update existing session status to "stopped" in DB
  3. Create new CogTestSession row (status="active")
  4. Instantiate CogTestEngine(session_id=new.id, concept=body.concept)
  5. register_engine(new.id, engine)
  6. Return `{"session_id": new.id, "concept": body.concept}`

**GET /cog-test/sessions** — List sessions
- Auth: `Depends(get_current_user)`
- Query DB for all CogTestSession where user_id=current_user.id, order by created_at desc
- Return list of `{"id", "concept", "status", "created_at"}`

**GET /cog-test/sessions/{session_id}/stream** — SSE stream
- Auth: `Depends(get_current_user_from_query)` (JWT via ?token= query param)
- The stream endpoint function signature must declare TWO separate db parameters:
  1. `current_user: User = Depends(get_current_user_from_query)` — this dep has its own internal db session for auth; do NOT reuse it
  2. `db: AsyncSession = Depends(get_db)` — this is the explicit, separate session passed to engine.run(db)
  Example signature:
  ```python
  async def stream_session(
      session_id: str,
      current_user: User = Depends(get_current_user_from_query),
      db: AsyncSession = Depends(get_db),
  ):
  ```
- Verify session belongs to current_user (404 if not found, 403 if wrong user)
- Get engine: `engine = get_engine(session_id)` — if None, return 404 (session not active)
- Return `EventSourceResponse(engine.run(db))` — pass the explicit `db` parameter, not any session from inside the auth dep

**POST /cog-test/sessions/{session_id}/turns** — Submit user turn
- Auth: `Depends(get_current_user)`
- Body: `{"text": str}`
- Get engine, verify ownership, call `await engine.submit_user_turn(body.text)`
- Return `{"status": "ok"}`
- Return 404 if engine not found (session not active/streaming)

**POST /cog-test/sessions/{session_id}/stop** — Stop session
- Auth: `Depends(get_current_user)`
- Get engine, verify ownership, call `await engine.stop()`
- Update CogTestSession.status = "stopped" in DB
- Return `{"status": "stopped"}`
- If engine not found: still update DB status and return 200

Use `from app.core.database import get_db` (or wherever AsyncSession dependency lives — check existing imports in the file). Use `from sqlalchemy import select` for queries. Use `from app.api.deps import get_current_user_from_query`.
