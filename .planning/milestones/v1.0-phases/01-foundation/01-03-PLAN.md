# Plan 03: Agent Prompts + Output Schema

## Objective
Design and validate the Guide and Challenger agent system prompts with Socratic contract enforcement. Define the structured JSON output schema for blind spot extraction. Create a parser utility that extracts analysis JSON from agent responses.

## must_haves
- Guide system prompt: ends every response with a question, never gives direct answers
- Challenger system prompt: acknowledges what learner got right before questioning
- Both prompts produce Chinese conversational output
- JSON analysis schema defined for blind spot extraction
- Parser utility extracts analysis block from agent response text
